下载  Windows6.1-KB2506143-x64.msu 安装powershell 3.0
通过powershell 3.0 安装 Chocolatey
通过Chocolatey安装ruby
通过ruby的gem安装bundler
```
C:\>gem install --local C:\rubygems-update-2.6.7.gem
C:\>update_rubygems --no-ri --no-rdoc
```
